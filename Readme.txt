Task 1
Da linha 215 a 220 encontram-se as variáveis que são utilizadas como input, 
icod para escolher o método de análise (1- Newton; 2- Broyden), t1 para escolher o valor de teta1, 
t2 para escolher o valor de teta2 e tolm para escolher a tolerância máxima. 
Depois de preenchidos esses valores, basta rodar o código.

Task 2
Da linha 150 a 164 encontram-se as variáveis que são utilizadas como input,
icod para escolher dentre os 4 principais métodos (1- Encontrar raiz; 2- Calcular integral; 
3- Calcular derivada ar partir de deltax fornecido; 4- Estimar derivadas pela extrapolação de Richard),
icod2 para escolher o método a ser utilizado em cada grande método (1,1 - Newton Raphson; 1,2 - Bisseção; 
2,1- Integral Polinomial; 2,2- Quadratura; 3,1- Passo a Frente; 3,2- Passo Atrás; 3,3- Diferença Central; 4,1- Richard),
c1, c2, c3 e c4 como constantes, tolm como tolerância máxima, a e b sendo os pontos que formam o intervalo
tanto no método 1 quanto no 2, x, dx e dx2 são parâmetros referentes aos métodos 3 e 4. 

Task 3

m, c e k como constantes da EDO e a1, a2, a3, w1, w2, w3 como constantes função F(t).
